Music System for Unity - Requirements
--------------------------------------
MusicXML => Supports MusicXML 3.x or above. For a better visual result we recommend MusicXML 4.x or above
Sibelius => Dolet 8 or higher for exporting MusicXML 
Accompaniment => OGG format 44.1kHz or higher
Midi => Type 1 - Multi-track, Tick ResolutionPPQN = 960
